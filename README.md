## 📘 About az104-cert-buddy

az104-cert-buddy is an AI-powered study tool created to help you prepare for the AZ-104 Microsoft Azure Administrator certification. It generates exam-like practice questions, lets you check your answers, and provides hands-on labs in the Azure portal. These labs reference real Microsoft Learn content to help you gain practical skills.

This application is designed for anyone studying for the certification, whether you are new to Azure or looking to refresh your knowledge. You don’t need technical skills to use it. The tool guides you through practice questions and lab exercises, so you can build your confidence before the exam.

## 🔧 How to Use az104-cert-buddy

### Open the App

1. Click the az104-cert-buddy icon on your Desktop or in your Start menu.
2. When the app opens, you will see options for practice questions and labs.

### Practice Questions

- Select the **Practice Questions** tab.
- Choose a topic or take a full mock test.
- Answer questions and submit them by clicking **Submit**.
- The app will show your score and explain correct answers.

### Azure Portal Labs

- Select the **Labs** tab.
- You will see a list of labs covering key skills for the AZ-104 exam.
- Each lab will include step-by-step instructions.
- Some labs will open links to the Azure portal where you can perform tasks.
- You need to sign in with your Microsoft Azure account to complete these labs.
